# Automated Network Request Management in ServiceNow

## Project Overview
This project automates network-related service requests such as VPN access, WiFi access, LAN port activation, and firewall changes using ServiceNow.

## Features
- Service Catalog Request Form
- Approval Workflow
- Auto Task Assignment
- Notifications
- Request Tracking
- Reports Dashboard

## Phases
1. Requirement Analysis & Planning
2. Backend Development & Configurations
3. UI/UX Development & Customization
4. Testing & Security
5. Deployment & Documentation

- ServiceNow
- GitHub
